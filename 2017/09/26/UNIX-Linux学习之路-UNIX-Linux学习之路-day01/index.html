UNIX/Linux学习之路_day01

发表于 2017-09-26 | 分类于 操作系统管理

本系列文章是我在2017年9月开始阅读的系列图书

《UNIX and Linux System Administration Handbook(4th Edition)》UNIX/Linux 系统管理手册

所做的阅读笔记，借此记录所学，温故而知新。

本书共分为 三部分

大致上将 UNIX/Linux 系统管理分为以下几大类

  • 基本管理
  • 网络管理
  • 其他管理

基本管理

  • 入门
  • scripting 和 shell
  • 引导和关机
  • 访问控制和超级权限
  • 进程控制
  • 文件系统
  • 添加新用户
  • 存储
  • 周期性进程
  • 备份
  • 系统日志和日志文件
  • 软件安装和管理
  • 驱动程序和内核

网络管理

  • TCP/IP
  • DNS
  • 路由
  • 网络硬件
  • 域名系统
  • 共享文件系统
  • 电子邮件
  • 网络管理和调试
  • 安全
  • Web主机托管

其他管理

  • 虚拟化
  • X窗口系统
  • 打印
  • 数据中心
  • 绿色IT
  • 性能分析
  • 通过 Windows 分析
  • 串行设备和串行终端
  • 管理、政策与政治

浏览全书，基本上了解了UNIX/Linux管理的知识与技能图谱，为了充分的学习和熟悉这些知识，我将在接下来的每天，写作一章节的阅读笔记，记录学习。
